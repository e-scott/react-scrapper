# react-scrapper

This is a scrapper project that scrapes the site http://reactkungfu.com

To get the scrape, you will need to use the /scrape in the url.

This project uses:

<ul>
    <li>Node.js</li>
    <li>express</li>
    <li>request</li>
    <li>morgan</li>
    <li>body-parse</li>
    <li>jQuery</li>
    <li>bootstrap</li>
    <li>mongoose</li>
    <li>MongoDB</li>
</ul>
